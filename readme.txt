Jeu des petits chevaux
MEZIANE GHILAS - MENAA ABDERREZAK
-----------------------------------------------------------------
Le jeu des petits  chevaux est un jeu de plateau  qui se joue
de 2 à 4joueurs.
-----------------------------------------------------------------
pour créer l'executable il suffit de taper:
"make jeu" dans le terminal
et d'ensuite le lancer en tapant "./jeu" toujours dans le terminal
Il faut biensur au préalable se placer dans
le dossier ou se trouve les fichiers source
